git init [nombre -opcional]

para crear el repo local, opcionalmente puedo crear la carpeta.

git status 

Obtener el estado del repo: wWorking, Staging o Repository.


Saging
git add .    // Agrego todo
git add mi_archivo.ext   // Agrego solo el archivo


Repository
git commit -m "mensaje descriptivo del nodo"


Pra ver los commit's 
git log --oneline  //ver cada commit en una sola linea
git log --all // para ver todos los commit's 

Para cambiar de commit (tambien para ramas)
git checkout[numero de commit]
